### PILHAS:

#### **AULA DIA 10 - ESTRUTURA DE DADOS** 📅

- **Contrato** 📝  
  - **C** = **CREATE** (criar) ➕  
  - **R** = **READ** (ler) 📖  
  - **U** = **UPDATE** (atualizar) 🔄  
  - **D** = **DELETE** (apagar) 🗑️  
  - **S** = **SORT** (ordenar) 🔢

- **Tipos de Dados Numéricos** 🔢:
  - **INTEIROS**:  
    - unsigned int; // tinyint, smallint, int, long, LONGINT, BIGINT (SQL) 🔢

  - **PONTOS FLUTUANTES**:  
    - **FLOAT**, **DOUBLE**  
    - Ponto fixo: **NUMERIC** ou **DECIMAL** (SQL) 💡

- **String** 💬  
  - **string** = "ifba"  
  - **char** = 'A'  
  - **Lógico** = **boolean**, **bool**  
  - Pessoa: **pessoa["ana"]**

- **Vetores e Arrays** 🧮:
  - Vetor, array, matriz unidimensional  
  - Arrays ordinais  
  - Arrays associativos

- **Operações Principais** 🔧:
  - **Empilhar** = **push** ⬆️  
  - **Desempilhar** = **pop** ⬇️  
  - **Topo** = **topo** 🔝

- **Operações Auxiliares** 🛠️:
  - **Está cheia** = **isFull** (overflow) ❌  
  - **Está vazia** = **isEmpty** (underflow) ⚠️  
  - **Imprimir** = **print** 🖨️
